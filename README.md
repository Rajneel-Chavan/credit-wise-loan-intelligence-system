# Credit-Wise-Loan-Intelligence-System

AI-powered loan approval system using machine learning with interactive dashboard and real-time predictions.

## Features
- Loan approval prediction using ML
- Multiple model comparison
- Interactive dashboard with graphs
- Simple and clear decision output

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib / Seaborn

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
